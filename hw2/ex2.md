### Параметры обучения.
В представленной для изучения учебной нейросети Tensorflow есть 4 предустановленных датасета.
Экспериментирование с различными настройками и архитектурой показало ограниченность используемых данных для точного определения влияние каждого параметра.
Добавление в модель резуляции с разной скоростью чаще ухудшали работу нейросети.
Потому для данных Circle, Exclusive or и Gaussian выбраны следующие параметры :
- Problem type - Classification
- Regularization - none
- Regularization rate - 0
- Learning rate - 0.3
- Activation - Выпрямленная линейная функция активации ReLU, либо гиперболической функцией активации касательной tanh ,которая так же показала хорошую эффектвиность.
- FEATURES - выбраны функции х1 и х2
- Количество скрытых слоев - ументшено до одного, состоящего из 4 элементов.

Для датасета Spiral используются те же самые настройки, но для достижения приемлемой эффективности добавлены функции sin(X1) и sin(X2). Также увеличено количество элементов в скрытом слое до шести.
Более эффективным Activation являются Tanh и Sigmoid.

### Скорость обучения нейронной сети.

При  заданных параметрах удалось достичь эффективности и величины ошибки меньше 0.03 за 20 эпох обучения для первых трех датасетов.
А для датасета Spiral за 200 эпох/циклов обучения.

### Выводы

Данный инструмент интересен и полезен для ознакомления с принципами работы нейронных сетей. 
Однако поверхностное знакомство не дает ответов на то как достигается конкретный результат.
Более того - запуск одних и тех же тестов с идентичными параметрами - дасто дает раздичные результаты. Особенно тогда когда параметры подобраны не оптимальны.
Конкретных запуск дает приемлемый результат за 200 циклов обучения. Повторный же запуск может вообще не достичь приемлего Test loss даже за 2000 циклов обучения.
Такие результаты вызывают сомения в эффективности применения подобного подхода в практических задачах.
В такой трактовке настройка нейросетей превращается в исскуство, а не в технологию.
